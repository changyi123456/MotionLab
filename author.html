MotionLab Project

MotionLab 專案

MotionLab 2D / 3D Tracker 是針對物理課堂、探究與實作、科展研究所設計的影片分析工具。 專案整合舊有 Tracker 資源、DLTdv8 相關概念與課堂操作需求，讓使用者能更快完成校正、 追蹤、圖表檢視、資料輸出與 3D 軌跡分析。

目前 2D 與 3D 可攜版已放在 GitHub Releases；開源資料與 source package 先維持未公開， 待正式整理後再釋出。
